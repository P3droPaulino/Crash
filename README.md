
## Instalação

• Instalação(requer [git](https://gitforwindows.org/)):

```bash
  $ git clone https://github.com/davideduardotech/Crash.git
  $ cd Crash
  $ pip install -r requirements.txt
  $ python Crash.py
```


## Banco de Dados(MongoDB)
